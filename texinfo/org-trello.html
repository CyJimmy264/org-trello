3 Using org-trello

This section describes how to use org-trello after connecting an org-file to a board of Trello.com.

3.1 Summary of Key-Bindings

All default org-trello key-bindings use the “C-c o” prefix.

Type “C-c o C-h” to display help for all key bindings with this prefix.

KeysCommand
C-c o C-hDisplay help on all key-bindings with C-c o prefix.
C-c o vDisplay the version of org-trello.

3.2 Synchronization Functions

This section describes the synchronization functions.

org-trello-sync-buffer

Type “C-c o s” to sync the entire buffer TO the Trello board.
Type “C-u C-c o s” to sync the entire buffer FROM the Trello board.
